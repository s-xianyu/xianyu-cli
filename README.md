# Direct use of
```
$ npx s-xianyu-cli init myproject
```

# Local Installation and use
```
$ npm i s-xianyu-cli -g

$ xy init myproject
```

## upgrade
Check the new version is available or not:
```
$ s-xianyu-cli upgrade
```
